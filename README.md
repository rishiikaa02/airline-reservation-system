# Airline Reservation System

## Overview
Airline Reservation System is a Java-based application designed to manage flight booking and reservation operations for passengers.

## Features
- Domestic and International flight booking
- Passenger information management
- Ticket generation
- Login authentication system
- Flight selection interface
- Booking confirmation and ticket printing

## Tech Stack
- Java
- Object Oriented Programming (OOP)
- Java Swing / GUI Components

## Purpose
Developed as an academic project to understand real-world reservation workflows, object-oriented programming concepts, and GUI-based application development.
